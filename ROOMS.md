# Jet Set Willy Room List with Transport Codes

copied and updated for this codebase from

[https://skoolkit.ca/disassemblies/jet_set_willy/tables/rooms.html](https://skoolkit.ca/disassemblies/jet_set_willy/tables/rooms.html)

To transport around the maison (careful where you land):

* Go to the floor of First Landing, type `writetyper`, and a half-blue Maria
will show up to indicate that transportation mode has been activated.

* After that, press the teleport code and enter (at the same time, two keys or
possibly three if you're using shift), and you will transport.

* Once activated, half-blue Maria will be with you until you exit the game, so
you don't have to keep running to the first landing every time you want to test
something.
